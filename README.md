# Due Date Calculator

### Installation using your favorite package manager
```bash
# npm
npm install

# pnpm
pnpm

# yarn
yarn
```

### Run tests
```bash
# npm
npm run test

# pnpm
pnpm test

# yarn
yarn test
```

### Run code coverage
```bash
# npm
npm run coverage

# pnpm
pnpm coverage

# yarn
yarn coverage
```
